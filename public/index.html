One panel to rule every server

From the first boot to scaling across nodes, Cloud Panel ships with the features you'd expect from premium hosting — for free.

⚡

Real-time console

Stream stdout/stderr live over WebSockets with ANSI colors, log replay, and a command box with history.

📈

Live resource stats

True CPU, memory and disk usage sampled per server and drawn as smooth sparkline graphs.

🧩

One-click eggs

Create a server and Cloud Panel downloads the real jar (Paper, Purpur, Fabric, Velocity…) and accepts the EULA.

🔑

Per-server SFTP

Connect with FileZilla, WinSCP or VS Code on port 5657 — files scoped safely to each server.

🗂️

Built-in file manager

Browse, edit, rename and delete server files right from the browser, with path-traversal protection.

🛡️

Secure by design

JWT auth, bcrypt hashing, first-run setup wizard and zero default passwords.

🖥️

Nodes & allocations

Manage multiple nodes, locations and IP:port allocations from a slick admin console.

🗄️

Real database

Durable, transactional SQLite under the hood — with an automatic JSON fallback so it always runs.

☁️

One-command install

On your own VPS, a single script installs Node, Java, a systemd service and your firewall — then prints your admin login.

👥

Subusers & teams

Invite others to a server with granular permissions — console, files, power, backups, databases & more — enforced everywhere.

⏰

Scheduled tasks

Cron-style schedules for nightly restarts, timed backups and automated commands — set it and forget it.

🔐

Two-factor auth

Authenticator-app 2FA with QR enrollment and one-time recovery codes. Your TOTP secret never leaves the server.

🗄️

Per-server databases

Provision real MySQL/MariaDB databases per server against admin-managed hosts — the classic Pterodactyl feature.

🧩

Modrinth browser

Search Modrinth and one-click install plugins & mods — plus a one-click modpack egg that installs the loader for you.

🟢

Live player list

See who's online, parsed straight from the console, and kick or ban with a single click.

📊

Historical metrics

Per-minute CPU, RAM, disk & uptime history with real graphs — not just live sparklines.

🌐

Public status pages

A shareable, login-free status page per server with live status, player count and uptime.

Live control

Watch & command your servers in real time

The console streams every log line the instant it happens. Send commands, restart, or kill a runaway process — all without touching SSH.

  • WebSocket streaming with ANSI colors & log replay
  • Start / Stop / Restart / Kill with graceful shutdown
  • Crash detection & auto status updates

Up & running fast

Live in three steps

On a fresh Ubuntu/Debian VPS, one command does the whole install — then you create your admin in the browser.

1

Clone

Pull the repo onto your server.

git clone https://github.com/Notbangbang-dev/cloud-panel.git

2

Install

One script installs Node, Java, the service & firewall.

sudo bash scripts/install.sh

3

Set up

Open the panel, create your admin, and deploy your first server.

http://your-ip:8080

Questions

Frequently asked

Is Cloud Panel really free?
Yes — Cloud Panel is open source under the MIT license. Self-host it anywhere with no license keys, no limits, and no phone-home.
What do I need to run it?
Node.js 18+ for the panel itself. For Minecraft servers you'll also want Java — the VPS installer adds Temurin automatically. The demo egg needs nothing but Node.
Which servers can it run?
27 templates out of the box. Auto-installed: Paper, Purpur, Folia, Fabric, Quilt, Pufferfish, Leaf, Vanilla, Forge, NeoForge, Sponge, Velocity, Waterfall, BungeeCord and Geyser — plus one-click Modrinth modpacks and SteamCMD games (Rust, Valheim, CS2). Upload-your-files: Spigot (use Paper for auto), Minecraft: Bedrock, PocketMine-MP and Terraria, plus generic Java, Node.js and Python apps. Adding your own egg takes a few lines.
Can I share a server with my team?
Yes — invite other accounts as subusers with granular permissions (console, power, files, backups, schedules, databases, players and more). Secure accounts with authenticator-app 2FA, and provision real MySQL/MariaDB databases per server.
Are there default passwords?
No. On first run you create your administrator through a setup wizard (or the CLI). There are no built-in accounts to forget about.
Is there a Discord bot?
Yes! The companion Cloud Panel bot brings full moderation, a ticket system with HTML transcripts, welcome & leave messages, and autorole — 27 beautifully themed slash commands. Join the Discord to add it.
How is this site hosted?
This very page is a static site hosted on Vercel. Cloud Panel itself is the open-source app you self-host — see the docs to deploy it on your own VPS.
